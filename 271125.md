### Today I Learned ~
1. WHERE can't handle aggregation functions like MAX()
so you can use SUBQUERIES instead! 
~~~
-- wrong way to use
WHERE QUANTITY = MAX(QUANTITY);
-- correct way (use subquery)
WHERE QUANTITY = (SELECT MAX(QUANTITY) FROM PETRESQUE);
~~~
